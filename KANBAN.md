# Kanban Board

Bu projede görev yönetimi için Github Projects kullanılmıştır.

## Kolonlar
- Todo
- In Progress
- Done

## Görevler
- Product ve PerishableProduct sınıflarının oluşturulması
- Inventory ve StockManager geliştirilmesi
- Unit testlerin yazılması
- UML diyagramlarının hazırlanması
- Raporun tamamlanması

Görevler proje sürecinde Todo → In Progress → Done aşamalarından geçirilmiştir.
